# Secondary School Spaced-Repetition Error Log System

## Project Purpose
A serverless web application designed for Hong Kong secondary school mathematics students to log their mistakes (via photo uploads) and review them using a spaced-repetition algorithm (1, 3, 7, 14 days). Teachers have a dedicated dashboard to monitor high-risk concepts and individual student performance.

## Features
- **Role-based Access Control**: Admin, Teacher, and Student roles driven by a Google Sheets whitelist.
- **Photo Uploads**: Direct to Google Drive using HTML5 Canvas / Base64.
- **Spaced Repetition**: Automated checkpoints without email spam.
- **Data Analytics**: Teacher dashboard for error type and topic analysis.
- **Mobile First**: Built with Tailwind CSS, perfectly responsive for classroom use.
- **Google Sites Ready**: Easy deployment via Apps Script Web App iframe.
